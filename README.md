# Maya Photography Portfolio

A separate-page static photography portfolio for GitHub Pages.

## Pages
- `index.html` — Maya / personal work
- `gigs.html` — gigs and sessions
- `about.html` — about page
- `editorial.html` — editorial gallery
- `features.html` — features and commissions
- `contact.html` — contact page

## Update photos
Edit `gallery-data.js` and replace sample `src` values with Cloudinary URLs.

Recommended Cloudinary transformation:
`f_auto,q_auto,w_1600`

Example:
`https://res.cloudinary.com/YOUR_CLOUD_NAME/image/upload/f_auto,q_auto,w_1600/shamina-gallery/maya/photo-name.jpg`
